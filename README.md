# ssafy

## Project setup
```
npm install 
npm install -g yarn
npm install -g @vue/cli
npm install -g vue
npm install -g firebase-tools
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
